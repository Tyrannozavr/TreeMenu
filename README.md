# TreeMenu

this project is an implementation of a tree menu through django-tag

to start project run:
```shell script
git clone https://github.com/Tyrannozavr/TreeMenu
```

```shell script
cd TreeMenu
```

```shell script
pip install -r requirements.txt
```

```shell script
python3 manage.py runserver
```
